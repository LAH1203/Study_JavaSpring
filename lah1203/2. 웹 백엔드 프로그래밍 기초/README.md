##### Apache Tomcat이란?
```
Apache Tomcat은 Apache Software Foundation, ASF에서 개발한 세계에서 가장 많이 사용되는 WAS이다.
자바를 이용하여 작성된 웹 어플리케이션은 WAS가 있어야만 실행할 수 있다. 이 때 가장 많이 사용되는 것이 바로 Apache Tomcat이다.
```
